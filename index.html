Overview

The SoilSensorEmail System is a Raspberry Pi-based solution for monitoring plant soil moisture and sending email alerts when watering is required. It combines a soil moisture sensor (FC-28), Python’s smtplib for notifications, and a Tkinter GUI for real-time interaction. Developed with Agile principles, the project is managed on GitHub.

Features

  • Monitors soil moisture via a digital sensor on GPIO 4.
  • Sends email alerts at scheduled times (e.g., 07:00, 10:00) for dry soil.
  • Provides a Tkinter GUI for status, schedules, and email tracking.
  • Includes diagnostics for sensor and email functionality.
  • Supports a fallback mode for GUI use without hardware.

Setup

Hardware

  • Connect FC-28 sensor: VCC to 5V, GND to GND, DO to GPIO 4.
  • Insert sensor into soil and adjust potentiometer.

Software

  • Install libraries: 
    sudo apt update
sudo apt install python3-rpi.gpio
pip install tk
  • Clone repository: 
    git clone https://github.com/Nickory/Project2_2025.git
  • Configure email in SoilSensorEmailGUI.py.

Run

python3 SoilSensorEmailGUI.py

Development

Built using Scrum with a two-week sprint, the project involved user story planning, sensor/email integration, GUI development, and Git-based version control.

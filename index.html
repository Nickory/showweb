Overview

The SoilSensorEmail System is a Raspberry Pi-powered solution for monitoring soil moisture and sending email alerts for plant watering needs. It integrates an FC-28 sensor, Python’s smtplib for notifications, and a Tkinter GUI for real-time control. Built with Agile practices, the project is hosted on GitHub.

Features

  • Tracks soil moisture using a digital sensor on GPIO 4.
  • Sends scheduled email alerts (e.g., 07:00, 10:00) for dry soil.
  • Offers a Tkinter GUI for status, scheduling, and email monitoring.
  • Provides diagnostic tools for sensor and email functions.
  • Supports GUI operation without hardware in fallback mode.

Setup

Hardware

  • Connect FC-28 sensor: VCC to 5V, GND to GND, DO to GPIO 4.
  • Insert sensor into soil and adjust potentiometer.

Software

  • Install libraries: 
    sudo apt update
sudo apt install python3-rpi.gpio
pip install tk
  • Clone repository: 
    git clone https://github.com/Nickory/Project2_2025.git
  • Configure email in SoilSensorEmailGUI.py.

Run

python3 SoilSensorEmailGUI.py

Development

Developed using Scrum in a two-week sprint, the project involved user story planning, sensor and email integration, GUI creation, and Git version control.
